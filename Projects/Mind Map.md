[Link](mindmap://doc/d1783939913610-5)When using a password on a file or folder, ensure to encrypt the files and change the extension to .emd for the encrypted markdown. When the user opens an encrypted file, decrypt only the file with the used password in memory; the files on disk must remain encrypted. Do not store the passwords; use them solely for encryption and decryption purposes.

The \[encrypted]\(mindmap\://doc/d1783939913610-4) file is by default password protected and locked.

Ensure that the encrypted file does not appear in the \[vector]\(mindmap\://doc/d1783939913610-4) database.
