When using a password on a file or folder, ensure to encrypt the files and change the extension to .emd for the encrypted markdown. When the user opens an encrypted file, decrypt only the file with the used password in memory; the files on disk must remain encrypted. Do not store the passwords; use them solely for encryption and decryption purposes.

The encrypted file is by [default](mindmap://doc/d1783940260852-5) password protected and [locked](mindmap://doc/d1783940260852-2).

Ensure that the encrypted file [does](mindmap://doc/d1783940260852-3) not appear in the [vector](mindmap://doc/d1783940260852-3) database.
